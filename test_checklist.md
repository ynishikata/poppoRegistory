# テストチェックリスト

## 認証機能

### 新規登録
- [ ] メールアドレスとパスワードで新規登録できる
- [ ] 既に登録されているメールアドレスで登録しようとするとエラーになる
- [ ] パスワードが短すぎる場合、エラーメッセージが表示される

### ログイン
- [ ] 正しいメールアドレスとパスワードでログインできる
- [ ] 間違ったメールアドレス/パスワードでログインしようとするとエラーになる
- [ ] ログイン後、ユーザー情報が表示される

### ログアウト
- [ ] ログアウトボタンをクリックするとログアウトできる
- [ ] ログアウト後、認証が必要なページにアクセスできない

## ぬいぐるみ管理機能

### 一覧表示
- [ ] ログイン後、自分のぬいぐるみ一覧が表示される
- [ ] 他のユーザーのぬいぐるみは表示されない
- [ ] ぬいぐるみがない場合、適切なメッセージが表示される

### 新規登録
- [ ] 名前、種類、お迎え日、写真を入力して登録できる
- [ ] 写真なしでも登録できる
- [ ] 名前が空の場合、エラーメッセージが表示される
- [ ] 登録後、一覧に表示される

### 編集
- [ ] ぬいぐるみの情報を編集できる
- [ ] 写真を変更できる
- [ ] 他のユーザーのぬいぐるみは編集できない

### 削除
- [ ] ぬいぐるみを削除できる
- [ ] 削除後、一覧から消える
- [ ] 他のユーザーのぬいぐるみは削除できない

### 詳細表示
- [ ] ぬいぐるみの詳細ページが表示される
- [ ] 会話履歴が表示される
- [ ] 会話履歴を編集できる

## 会話機能

### 会話履歴の更新
- [ ] 会話履歴を入力して保存できる
- [ ] 保存後、詳細ページに反映される

### チャット機能
- [ ] 「話す」ボタンをクリックすると、LLMが応答を生成する
- [ ] 会話履歴に基づいた応答が生成される
- [ ] エラーが発生した場合、適切なエラーメッセージが表示される

## エラーハンドリング

### 認証エラー
- [ ] ログインしていない状態でAPIを呼び出すと、適切なエラーメッセージが表示される
- [ ] トークンの有効期限が切れた場合、適切なエラーメッセージが表示される

### バリデーションエラー
- [ ] 必須項目が空の場合、適切なエラーメッセージが表示される
- [ ] 無効なIDでアクセスした場合、適切なエラーメッセージが表示される

### サーバーエラー
- [ ] サーバーエラーが発生した場合、適切なエラーメッセージが表示される
- [ ] エラーログがサーバーに記録される
